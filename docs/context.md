# TOKYO Packaging Store
## Project Context

Version: 1.0

---

# IMPORTANT

Before writing any code, understand this business carefully.

This project is NOT a generic e-commerce platform.

This project is NOT Amazon.

This project is NOT Shopify.

This project is NOT Noon.

This project is NOT an ERP.

This project is NOT a POS.

This project is NOT a Marketplace.

This project is a specialized e-commerce platform designed specifically for selling packaging, disposable products, paper products and restaurant supplies.

Everything inside the application must serve this business domain.

---

# Project Vision

Build the most professional Arabic-first online platform for disposable packaging products in Egypt.

The platform should help restaurants, cafés, bakeries, companies and individuals purchase packaging products quickly with a modern shopping experience.

The experience should feel premium, trustworthy, clean and extremely easy to use.

# AI Development Rules

Always remember:

This is NOT a generic online store.

This is a specialized packaging commerce platform.

Never generate features related to fashion, electronics, groceries or marketplaces.

Every component must improve the packaging purchasing experience.

Always optimize for:

Restaurants

Cafés

Businesses

Bulk Buyers

Packaging Customers

Whenever uncertain,

prefer business usability over visual complexity.

Every future feature should support the packaging industry first.
