# Best Practices

- One Responsibility Per Schema
- Business Logic In Functions
- UUID Everywhere
- UTC Timestamps
- Strong Constraints
- ENUM For Status Fields
- RLS On Every Business Table
- Materialized Views For Heavy Reports
- No Business Data In auth Schema
- No Direct Writes To Managed Schemas
