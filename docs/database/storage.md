# File Naming Convention

Never

```
image.jpg

photo.png

new.webp
```

Always

```
<entity-id>-<timestamp>.webp
```

Examples

```
8ab1e2-product.webp

product-9837281.webp

receipt-20260718.webp

avatar-5f81c3.webp
```
