# Release Checklist

## Pre-Release
- [ ] Release plan completed
- [ ] All features in manifest are complete
- [ ] All fixes in manifest are verified
- [ ] Release candidate built and tested
- [ ] Smoke tests pass
- [ ] Regression suite passes
- [ ] Security scan passes
- [ ] Performance baseline verified
- [ ] Rollback procedure documented

## Release
- [ ] Staging deployment verified
- [ ] Production deployment executed
- [ ] Health checks pass
- [ ] Monitoring confirms operational status

## Post-Release
- [ ] Release notes published
- [ ] Stakeholders notified
- [ ] Post-release monitoring configured (24h)
