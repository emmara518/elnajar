# Naming Rules (Memory)

## Quick Reference
| What | Convention | Example |
|------|-----------|---------|
| Components | PascalCase | `ProductCard.tsx` |
| Hooks | camelCase + use | `useProducts.ts` |
| Utilities | camelCase | `formatDate.ts` |
| Constants | UPPER_SNAKE_CASE | `PAGE_SIZE` |
| Stores | camelCase.store.ts | `auth.store.ts` |
| Services | PascalCase.service.ts | `product.service.ts` |
| Files | kebab-case folders, PascalCase components | |
| Booleans | is/has/can/should prefix | `isLoading` |
| Event handlers | handle prefix | `handleSubmit` |
